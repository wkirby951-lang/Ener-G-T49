# Faster EFT Quick Reference Cheat Sheet

## Two-Minute Overview

Faster EFT is a streamlined, inquiry-based tapping approach that prioritizes **body-sensing**, **curiosity**, and **cognitive reframing** over repetitive phrases. It uses fewer tapping points per round but goes deeper.

---

### The Core Loop

```
1. FEEL IT → 2. TAP IT → 3. ASK → 4. NOTICE SHIFT → 5. REFRAME → REPEAT
```

---

### Faster EFT Protocol (5 Steps)

**Step 1: Tune In**
Close your eyes. Think of the issue. Where do you feel it in your body? Place your hand there.

**Step 2: Tap + Acknowledge**
Tap on the collarbone point and say: *"I notice this sensation in my [body part]. I'm not trying to change it. I'm just noticing."*

**Step 3: Inquire (The Key Difference)**
While tapping, ask open-ended questions:
- "What shape is this sensation?"
- "What color? What texture? What temperature?"
- "If it could speak, what would it say?"
- "What emotion is underneath?"
- "What belief is underneath that?"
- "How old is this feeling?"

**Step 4: Reframe**
Once you identify the belief, introduce a new possibility:
- "I've held this belief for a long time. It's familiar. But familiar doesn't mean true."
- "What if this belief isn't the whole truth?"
- "If I could choose a new belief, what would it be?"

**Step 5: Anchor**
Tap your new belief or set-point word through 4–8 points. Recheck the body. Repeat the loop if needed.

---

### Key Points (Just 4 — Not 9)

| Point | Use |
|-------|-----|
| **Collarbone (CB)** | Primary point — hold the felt-sense here |
| **Side of Eye (SE)** | For cognitive reframing and inquiry |
| **Under Eye (UE)** | For emotional processing |
| **Top of Head (TH)** | For integration and anchoring |

*You can add Karate Chop (KC) for setup if you prefer the classic opening.*

---

### Inquiry Questions (Cheat Sheet)

| Category | Questions |
|----------|-----------|
| **Sensation** | Where? What shape? Color? Texture? Temperature? Weight? |
| **Emotion** | What's the primary feeling? What's underneath that? |
| **Cognition** | What belief is attached? How old is this belief? Where did it come from? |
| **Metaphor** | If this sensation had a message, what would it be? If it was an animal/weather, what? |
| **Shift** | What changed? What's different now? What's here now? |
| **Possibility** | What would I like to feel instead? What would that feel like in my body? |

### The "Set Point" Reset

Your emotional set point is your nervous system's "normal" baseline.

**Signs of a high set point:**
- Always on edge / expecting something to go wrong
- Difficulty relaxing even in safe situations
- Your "calm" still feels tense

**How to reset with Faster EFT:**
1. Identify your current set-point word: "anxious," "vigilant," "numb," "driven"
2. Choose a new set-point word: "calm," "present," "safe," "ease," "open"
3. Tap on the collarbone while repeating the new word for 30 seconds
4. Do this daily for 21 days to build a new neural pathway

---

### Troubleshooting

| Issue | Solution |
|-------|----------|
| Nothing happened physically | Try a different point or a slower tapping pace |
| The sensation moved | Follow it — that's processing |
| Emotion came up strongly | Keep tapping, breathe, don't suppress |
| Can't find a "felt sense" | Start with a tension you can feel (e.g., jaw tightness) |
| Set point doesn't stick | Practice daily; it takes repetition to rewire |
